# SimCast Website

Marketing and landing page for [simcast.dev](https://simcast.dev) — the open-source iOS Simulator streaming tool.

## Tech Stack

- [Astro 6](https://astro.build) — static site generator
- [Google Fonts](https://fonts.google.com) — Geist, Geist Mono, Lora
- [@astrojs/sitemap](https://docs.astro.build/en/guides/integrations-guide/sitemap/) — auto-generated sitemap

## Getting Started

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # outputs to dist/
npm run preview    # preview the build locally
```

## Deployment

The site builds to static HTML in `dist/`. Deploy to any CDN — Vercel, Netlify, Cloudflare Pages, etc.
